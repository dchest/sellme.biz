Группа sellme-dev — девелоперский филиал sellme.biz. Туда я отправляю свои заметки по программированию и там мы обсуждаем, в основном, программирование под Mac.

Недавние полезные топики:

  • Шорткаты для Xcode и других Cocoa-приложений — отличный способ увеличить продуктивность.

  • Пример использования Quick Look view в своем приложении — с помощью приватного фрейворка недоступное становится доступным.

  • Лог git в календаре — скриптик для вывода лога коммитов git в календарь (например, в iCal, Google Calendar, Outlook...), чтобы следить за своей продуктивностью.

  • Crash reports — Vitaly Ovchinnikov делится фрейворком для отправки крэш репортов через HTTP.
