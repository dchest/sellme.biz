--- author: "dchest" date: 2010-01-14 04:28:36 +00:00 markup: markdown status: "Publish" title: "Если бы программы писали на Луне" --- -- Это наша программа. app = Application() -- А это кнопка. firstButton = Button:new{ -- У нее есть название, title = "Сказать привет", -- размеры, width = 160, height = 40, -- положение x = 10, y = 10, -- и действие. action = function () print "Привет, мир!" end -- Вот. } -- Это вторая кнопка, которая делает то же самое, secondButton = firstButton:new{ -- только называется по-другому title = "Опять сказать", -- и находится правее. x = 170 -- Вот. } -- Это окошко, у которого есть название, положение и размеры. window = Window:new{ title = "Программа для передачи приветов миру", x = 0, y = 0, width = 360, height = 60, -- А еще в этом окошке есть наши кнопки! subviews = { firstButton, secondButton } -- Вот. } -- Теперь мы скажем окошку показаться, window:show() -- а программе — запуститься. app:run() --[[ ![Луноход](/pics/lunokhod-demo.png) Это космические технологии. Но не rocket science. Жаль, что на Земле так не пишут. --]]
