--- author: "dchest" date: 2010-05-03 19:12:28 +00:00 markup: markdown status: "Publish" title: "Какими программами я пользуюсь" --- [Cyberduck][cyberduck] и [Interarchy][interarchy] для перекачки файлов по SFTP. Раньше пользовался только Cyberduck, но он *очень* медленный (не только интерфейс тормозной, но и собственно, функция закачки). Пробовал [Transmit](http://panic.com/transmit/) 3 и Interarchy, первый не устраивал (сейчас, правда, вышла крутая 4-я версия), второй стоил слишком дорого. Но тут попалась скидка на Interarchy (за $19.95 + бесплатно следующая версия, которая скоро выйдет) и перешел на него. Но Cyberduck до сих пор использую для Rackspace Cloud Files, потому что Interarchy не поддерживает этот сервис. Мечусь между [TextMate][textmate], [Emacs][emacs] и [MacVim][macvim], иногда [gedit][gedit] для всего (редактирование текстов, программирование вне Cocoa, заметки). До сих пор не определился с редактором. [Google Docs][googledocs] для совместного редактирования текстов и (очень редко) электронных таблиц. [Chrome][chrome] (c [FlashBlock][flashblock], [Google Docs PDF/PowerPoint Viewer][docsviewer], [Send using Gmail (no button)][sendusinggmail] и модифицированным [Keyboard Navigation](http://github.com/dchest/keyboard-navigation-for-chrome)) с недавних пор стал основным браузером, до этого пользовался Safari. Стоят еще [Firefox][firefox] и [Opera][opera] для тестирования (а первый для [Firebug][firebug], [YSlow][yslow] и [Page Speed][pagespeed], хотя в Chrome/Safari [новый Web Inspector][webinspector] уже может вполне заменить их). Для почты раньше использовал Mail, теперь полностью пересел на [Gmail][gmail] в браузере. [Tweetie][tweetie] для управления своими четырьмя аккаунтами в твиттере. [Cathodique][cathodique] для просмотра видео с YouTube, естественно, и [Mémoires][memoires] (иногда) для всяких мелких заметок (очень сложно его использовать по полной программе, потому что он у меня всегда в режиме дебаггинга и создания его скриншотов и скринкастов). [Google Calendar][googlecalendar] (раньше использовал iCal), чтобы помечать важные прошедшие даты и следить за сроками (например, тогда-то отдал мак в сервис) и [смотреть когда следующая гонка F1][f1] . Календарем "на будущее" очень редко пользуюсь, а to-do-шки совсем не веду. [Acorn][acorn] для рисования растровых картинок. Иногда он достает багами и я запускаю [Gimp][gimp]. В [Inkscape][inkscape] рисую иконки и логотипы ([логотип Coding Robots](http://www.codingrobots.com/images/codingrobots_icon_64.png) и иконки [BlogJet](http://www.codingrobots.com/press/blogjet256x256.png), [Mémoires](http://www.codingrobots.com/images/memoires_128.png) и [Cathodique](http://www.codingrobots.com/images/cathodique_128.png) рисовал именно в нем). В [Kindle for Mac][kindle] читаю книжки (синхронизируется с Kindle for iPhone, плюс еще стоит [Stanza][stanza] для закачки книг *не с амазона* на Stanza for iPhone). [MarsEdit][marsedit] для постинга в блоги. [Reggy][reggy] для проверки регулярных выражений. [The Unarchiver][theunarchiver] для распаковки экзотических архивов. [Transmission][transmission] для торрентов. [VirtualBox][virtualbox], чтобы запускать Windows и Linux. [VLC][vlc] для просмотра видюшек, у которых громкость слишком маленькая для QuickTime Player. [GitX][gitx] для просмотра истории репозиториев (остальные действия с git — через командную строку). [Clyppan][clyppan]. Недавно понял, что неплохо бы иметь историю буфера обмена. Пользуюсь виджетом [Minutes][minutes], когда готовлю еду. Очень красивый и простой таймер. iTunes. Комментарии не нужны. Address Book, синхронизирующаяся с Gmail, MobileMe и iPhone. Фотки храню во [Flickr][flickr] и [Picasa Web][picasaweb], некоторые в iPhoto. [Audacity][audacity], использую когда нужно записать звук, не ожидая пока GarageBand запустится. GarageBand для микширования лупов и сэмплов для звуковой дорожки в скринкасты (там очень обширная библиотека royalty-free сэмплов). В качестве калькулятора и конвертера валют использую Google. Для сложных вычислений — python или irb в консоли. [Xcode][xcode] для программирования под Мак. [ScreenFlow][screenflow] для записи и редактирования скринкастов. Программа дорогая, но лучшая из всего, что видел. [HexFiend][hexfiend], когда надо посмотреть бинарный файл. *Обновление:* забыл про Terminal и консольные программы! Из того, что не идет с Mac OS X: * [homebrew](http://github.com/mxcl/homebrew) (хорошая альтернатива MacPorts) * [git](http://git-scm.com/) * [optipng](http://optipng.sourceforge.net/) * [autojump](http://wiki.github.com/joelthelion/autojump/) (альтернатива "cd" :) * tree * [osx-trash](http://www.dribin.org/dave/osx-trash/) (альтернатива "rm") * wget (для создания зеркал, для остального — curl) * [jekyll](http://github.com/mojombo/jekyll) (для некоторых сайтов) Еще [Growl](http://growl.info/), [Perian](http://www.perian.org/), [Flip4Mac](http://www.telestream.net/flip4mac-wmv/overview.htm), [XiphQT](http://xiph.org/quicktime/) и [x264Encoder](http://www003.upp.so-net.ne.jp/mycometg3/) (кодирует видео лучше встроенного).
